# 🎧 Zero Two Bongo

![Tkinter](https://img.shields.io/badge/GUI-Tkinter-ff69b4?style=for-the-badge)

Мини‑кликер по Zero Two: кликай, прокачивай множитель, открывай скины и смотри на бонго‑анимацию.  
Игра написана на Python + Tkinter, без интернета и регистрации. [web:258]

---

## 🚀 Функции

- 💥 **Кликер‑геймплей**  
  Нажимай кнопки или клавиши — увеличивай счёт и качай множитель.

- 🛒 **Магазин улучшений**  
  - покупка множителей (x2, x4 и далее);  
  - улучшение автокликера;  
  - покупка альтернативного ALT‑скина;  
  - ускорение анимации.

- 🎨 **Система скинов**  
  - MAIN‑скин доступен сразу;  
  - ALT‑скин появляется в инвентаре **только после покупки**;  
  - переключение скинов через инвентарь.

- 💾 **Система сохранений**  
  - ручная кнопка **«Сохранить»** внизу экрана;  
  - автоматическое сохранение при выходе в меню и закрытии игры;  
  - настройки окна (разрешение) тоже сохраняются.

- 🛡️ **Защита сохранений и от читов**  
  - основной прогресс хранится в защищённом `savegame.txt` с подписью (checksum); [web:239][web:240]  
  - если файл меняют руками — игра просто **не загрузит** такие данные; [web:243][web:244][web:245][web:246]  
  - анти‑чит лимиты: слишком огромные значения счёта/множителя/скорости анимации считаются читерскими и сбрасываются. [web:244][web:247]

## 📥 Установка и запуск

### Вариант 1 — готовый `.exe`

1. Скачай последний релиз из раздела **Releases** на GitHub. [web:259]  
2. Распакуй ZIP в любую папку.  
3. Запусти файл `ZeroTwoBongo.exe`.

При первом запуске игра сама создаст свои файлы:

- `savegame.txt` — защищённое сохранение;  
- `settings.txt` — настройки окна.

Твои сохранения **не входят** в ZIP‑релиз, так что при обновлениях ты не шаришь свои данные другим людям.

---

### Вариант 2 — запуск из исходников

Требования:

- Python 3.9+  
- Библиотека **Pillow** (`PIL`) [web:258]

Установка зависимостей:

```bash
pip install pillow
```

Запуск:

```bash
python zero_two_bongo.py
```

Убедись, что рядом с `zero_two_bongo.py` лежат:

- `zero_two.gif` — основной GIF;  
- `zero_two_alt.gif` — альтернативный GIF (для ALT‑скина).

## 🎮 Управление

- **Hit! кнопка** — клики мышью.  
- **Любая клавиша клавиатуры** — тоже даёт клик.  
- **Shop** — открыть магазин улучшений.  
- **Сохранить** — вручную записать прогресс в `savegame.txt`.  
- **⚙ Настройки** — выбор разрешения окна (640×640, 800×600, 1024×768).  
- **Меню** — вернуться в главное меню.  
- В главном меню кнопка **«Выход»** закрывает игру с сохранением.

## 🔐 Защита от читов и кибератак

Игра полностью офлайн и не использует сеть, так что никаких удалённых “кибератак” через неё провести нельзя. [web:249][web:252]

### Что защищено

- Прогресс (score, множитель, апгрейды, скины) хранится в одном файле `savegame.txt` в виде JSON + подпись. [web:239][web:240][web:243]  
- При загрузке игра:
  - пересчитывает подпись и сравнивает её с сохранённой;  
  - проверяет значения на адекватность (лимиты);  
  - при несоответствии считает файл читерским/битым и не грузит его. [web:244][web:245][web:246][web:247]

### Что всё ещё можно сделать

Любая одиночная офлайн‑игра теоретически может быть модифицирована продвинутыми читерами (декомпиляция, патч exe и т.п.), но для обычных игроков такая защита уже сильно усложняет накрутку. [web:243][web:244][web:245][web:246]
